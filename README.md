Разработка и тестирование матричного ядра, выполняющего пару матричного умножения-сложения (MMAC).

HDL часть
Ядро представляет собой умножитель 2x2 в точности int16. Данные передаются по шине axi4-lite, что является главным ограничением проекта (в угоду простоте).
Данное ядро можно подключить к MicroBlaze V (или другому софт-процессору с шиной axi4-lite).

 Регистры ядра: (операция D = A*B+C)

| Смещение | Назначение         |
|----------|-------------------|
| 0x00     | Ряд 0 матрицы А   |
| 0x04     | Ряд 1 матрицы A   |
| 0x08     | Ряд 0 матрицы B   |
| 0x0C     | Ряд 1 матрицы B   |
| 0x10     | Ряд 0 матрицы С   |
| 0x14     | Ряд 1 матрицы C   |
| 0x18     | Ряд 0 выхода D  |
| 0x1C     | Ряд 1 выхода D |
| 0x20     | Управляющий регистр |
| 0x24     | Регистр состояния  |

Управляющий регистр управляется двумя битами:
1)Бит 0 - Запуск умножения
2)Бит 1 - Игнорирование C при выставленной 1

Регистр состояния содержит 1 (бит 0) - сигнал valid (по факту готовность при 1) с вычислителя

Софтварная часть
Реализован пример программы в main.c, содержащий единчиное умножения блока 2x2, а потом и 32x32 блоками по 2 (итого 256 операций MMAC).

Проект был собран на плате microphase a7-lite на artix 7 a35t при помощи vivado 2024.2 и vitis 2024.2

В дальнейшем планируется сделать собственный псевдо-GPU с векторными и матричными операциями и своей памятью
